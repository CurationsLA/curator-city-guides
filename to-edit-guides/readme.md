## How to Work with These Files

### Creating a Pull Request (Recommended Workflow)

1. **Create a feature branch:**
   ```bash
   git checkout -b update/neighborhood-name-holiday-2025
   ```

2. **Create the proper directory structure:**
   ```bash
   mkdir -p winter-2025/neighborhood-name
   ```

3. **Move and rename the guide:**
   ```bash
   cp to-edit-guides/winter-2025_neighborhood_guide_Version2.md \
      winter-2025/neighborhood-name/neighborhood-guide.md
   ```

4. **Enhance the content:**
   - Verify all dates and event information

### Enhancement Checklist

When working on a guide, ensure it has:

- [ ] Accurate, current event dates and times
- [ ] Mix of well-known spots and hidden gems
- [ ] Proper source citations (inline references)
- [ ] Engaging, descriptive writing that captures the neighborhood's character
- [ ] Complete sections: Eat, Do, Stay, Shop, Takeaway
- [ ] Interactive map reference and mini calendar
- [ ] SEO meta description
- [ ] JSON-LD TravelGuide schema markup
- [ ] No typos or grammatical errors

### Collaboration Tips

- **Ask questions** - Not sure about something? Tag the team!
- **Iterate together** - What is working? What can we improve?
- **Share insights** - Know a great local spot? Add it!
